# 定数倍高速化
1. プロファイリングの結果(perf.data)を踏まえて、xxx.cppを高速化するアイデアを挙げてください。ロジックは基本的に変更せずに定数倍高速化をしたいです。同等の操作や管理を異なるロジックで行える場合のみロジックの変更が許可されます。

2. アイデアごとに元のファイルをコピーし、それぞれのアイデアを個別に適用して実装してください。ファイル名は、{元のファイル名}_xxとし、00から連番で、既存ファイルと重複しないようにしてください。
    - ここでスコア向上するか、イテレーションが増えているかをチェック。

3. 新たに元のファイルをコピーし、アイデア1,2,...を全て実装してください。   
    - 高速化は基本的に独立しているはずなのでマージしても良いはず。
